Выбор языка программирования
Для реализации проекта — был выбран язык Python. Ниже приведены ключевые аргументы в пользу этого решения, а также сравнение с другими популярными языками.
Почему Python?
1. Отличная поддержка Telegram Bot API 
Существуют зрелые, хорошо документированные библиотеки:
   - aiogram — современный асинхронный фреймворк, идеально подходящий для ботов с высокой отзывчивостью.
   - python-telegram-bot  — стабильная синхронная альтернатива с поддержкой вебхуков и long polling.

2.  Лучшие инструменты для веб-парсинга 
   Python — де-факто стандарт в области скрапинга:
   - Для статических сайтов: requests + BeautifulSoup.
   - Для динамических (как Avito): Selenium, Playwright, scrapy.
   - Богатая экосистема для обхода защит (user-agent rotation, прокси, задержки).

3. Интеграция с облачными сервисами* 
   Python отлично работает с Yandex Cloud, PostgreSQL, Redis, Object Storage и другими сервисами, планируемыми к использованию.

4. Большое сообщество и документация 
   При возникновении сложностей (например, при парсинге Avito) легко найти готовые решения, сниппеты и советы на Stack Overflow, GitHub и в Telegram-каналах.

 Вывод
Python обеспечивает оптимальный баланс между скоростью разработки, функциональностью и поддерживаемостью. Он позволяет в кратчайшие сроки создать MVP с возможностью дальнейшего масштабирования и интеграции с облачной инфраструктурой. Учитывая специфику проекта — интенсивный парсинг + работа с Telegram — Python является наиболее рациональным и эффективным выбором.
